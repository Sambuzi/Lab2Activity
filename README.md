# Lab 2 – Robot Navigation and Optimization

## Descrizione

L'obiettivo di questo laboratorio è **ottimizzare il comportamento di un robottino autonomo** affinché riesca a completare un percorso nel **minor tempo possibile**.

Durante il tragitto il robot deve:
- **Rilevare ed evitare ostacoli** presenti lungo il percorso.
- **Orientarsi verso una fonte luminosa** che rappresenta l'obiettivo finale.

Per svolgere questo compito il robot utilizza:
- **Sensori di prossimità**, per individuare ostacoli e prevenirne la collisione.
- **Sensori di luce**, per rilevare la direzione della sorgente luminosa.

L'algoritmo sviluppato deve quindi permettere al robot di:
1. Analizzare continuamente i dati provenienti dai sensori.
2. Evitare gli ostacoli presenti nell'ambiente.
3. Muoversi in modo efficiente verso la fonte luminosa.

## Obiettivo del laboratorio

Progettare e migliorare progressivamente il comportamento del robot per:
- **Ridurre il tempo necessario per raggiungere la fonte luminosa**
- **Ottimizzare il movimento evitando collisioni**
- **Sfruttare in modo efficace i sensori disponibili**

## Risultato atteso

Il robot deve essere in grado di **navigare autonomamente nell'ambiente**, evitare gli ostacoli e **raggiungere la sorgente luminosa nel minor tempo possibile**.
